# Frontend de la aplicación de reservas de hoteles

## Tecnologías

Angular2

## Instalando dependencias
 
```
$ npm install -g typings
$ npm install -g concurrently
$ npm install
```

## Iniciando la aplicación
```
$ npm start
```
